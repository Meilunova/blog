### 2. 导航菜单配置 (_config.anzhiyu.yml)

```yaml
menu:
```

### 4. 奶茶色系主题配置

```yaml
theme_color:
  enable: true
  main: "#D2B48C"          # 奶茶色主色调
  dark_main: "#8B7355"     # 深色模式奶茶色
  paginator: "#DEB887"     # 分页器颜色
  text_selection: "#F5DEB3" # 文字选中颜色
  link_color: "#CD853F"    # 链接颜色
  toc_color: "#D2B48C"     # 目录颜色
```
